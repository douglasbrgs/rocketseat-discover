## Font Family

-   Tipo de fonte de um elemento
-   Lista de fontes e ordem de prioridade

```css
p {
    font-family: 'Times New Roman', Times, serif;
}
```

-   serif (tem dobras nas letras)
-   sans-serif (sem dobras)

## Font Weight

-   Peso da fonte

```css
p {
    font-weight: bold;
}
```

## Font Style

-   O estilo da fonte

```css
p {
    font-style: italic;
}
```

## Font Size

-   O tamanho da fonte

```css
p {
    font-size: 18px;
}
```
